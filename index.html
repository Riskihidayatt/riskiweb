HTML

Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat struktur dan mengatur tampilan konten pada halaman web. HTML menggunakan elemen-elemen (tag) untuk mempresentasikan informasi, dan browser web akan menginterpretasikan tag-tag tersebut untuk menampilkan halaman web secara sesuai.

Contoh Kode HTML

Berikut adalah contoh kode HTML yang sederhana:

      <!DOCTYPE html>
      <html>
      <head>
        <title>Judul Halaman</title>
      </head>
      <body>
        <h1>Judul Utama</h1>
        <p>Ini adalah contoh paragraf.</p>
        <p>Ini adalah contoh paragraf lainnya.</p>
      </body>
      </html>

Tag Dasar HTML

Berikut adalah beberapa tag dasar yang umum digunakan dalam HTML:

  • <h1> - <h6>: Menandai judul atau heading pada halaman.
  • <p>: Menandai paragraf teks.
  • <a>: Membuat tautan (link) ke halaman atau sumber lainnya.
  • <img>: Menampilkan gambar di halaman.
  • <ul> dan <li>: Membuat daftar dengan tanda bullet.
  • <ol> dan <li>: Membuat daftar dengan angka.
  • <div>: Menandai suatu bagian atau kontainer di dalam halaman.
  • <span>: Menandai bagian teks yang ingin diberi gaya atau manipulasi tertentu.

CSS

Pengertian CSS

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan gaya elemen-elemen HTML pada halaman web. Dengan CSS, Anda dapat mengubah warna, ukuran, jenis font, tata letak, dan berbagai aspek visual lainnya dari elemen-elemen pada halaman web.

Contoh Kode CSS

Berikut adalah contoh kode CSS yang menerapkan beberapa properti:

      /* CSS Styles */
      body {
        font-family: Arial, sans-serif;
      }
    
      h1 {
        color: blue;
      }
    
      p {
        font-size: 18px;
      }
    
      .container {
        background-color: lightgray;
        padding: 20px;
      }
    
      .highlight {
        background-color: yellow;
        font-weight: bold;
      }

Properti Dasar CSS

Berikut adalah beberapa properti dasar yang umum digunakan dalam CSS:

  • color: Mengubah warna teks.
  • font-size: Mengubah ukuran teks.
  • font-family: Mengubah jenis font teks.
  • background-color: Mengubah warna latar belakang elemen.
  • padding: Menambahkan ruang padding di sekitar konten elemen.
  • margin: Menambahkan ruang margin di sekitar elemen.
  • border: Menambahkan garis batas di sekitar elemen.
  • text-align: Mengatur tata letak teks (misalnya, "center" untuk tengah, "left" untuk kiri, "right" untuk kanan).
  • font-weight: Mengubah ketebalan (boldness) teks.
